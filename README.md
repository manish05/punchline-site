# Punchline — live site

Static build of [Punchline](https://github.com/manish05/punchline): short jokes
mined from popular YouTube comedy and ranked by the "Most Replayed" laugh graph,
reach, repetition, and LLM funniness/clarity. One-liners + dad-jokes (Q&A).

Live: https://manish05.github.io/punchline-site/

(Source pipeline lives in the private `punchline` repo. This repo is just the
static frontend + exported `jokes.json`.)
